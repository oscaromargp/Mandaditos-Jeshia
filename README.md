## 🎨 Personalización

### Cambiar Color Principal
Edita `styles.css` y busca la sección `/* CSS Variables - Color System */`, luego cambia el valor `--primary-400` al color deseado.

### Cambiar Imágenes
1. Genera las imágenes usando los prompts proporcionados
2. Reemplaza los archivos en la carpeta `/images/`
3. Actualiza las referencias en `index.html`
